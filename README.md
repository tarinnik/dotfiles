# Dotfiles

## Installation

### i3

#### Dependencies

- feh: wallpaper
- scrot: screenshot
- xrandr: set monitors
- arandr: gui to generate xrandr command
- xdotool: enable numlock
- polybar
- playerctl: music control
- dmenu: program launch, prompts

#### Things to edit

- $terminal variable
- $m1 and $m2 monitor variables
- start programs on launch
